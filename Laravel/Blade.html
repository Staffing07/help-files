What is Balde?

Blade is a php templating engine designed to be able to display data as html.

You do not have to use blade in a Laravel project, yopu can use normal php but Blade does make things easier.

Example of Blade syntax

Simple if statement

@if($price > 15)

This recipe is expensive

@endif

The @ means we are using blade to do something.
