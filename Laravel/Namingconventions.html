Naming Conventions

These are not requirements but are best practice

Model

Recipe Model

Capitalise first letter

Controller & action

Whatever the model is called followed by controller

RecipeController

Route & Actions

/recipes | RecipeController@index | View = index

used to get all recipe records

/recipes/{id} | RecipeController@show | View = show

Used to show one record

/recipes/create | RecipeController@create | View = create

Used to view a form that will add a new recipe to the db

View

To get all records index

RecipeController@show - to show a single record
