Eloquent models

Rather than manually typing SQL, Laravel uses models that it then wraps up in SQL code.

Each model can have several code methods to add, edit or remove data.

Creating an eloquent model

php artisan make:model Modelname

php artisan make:model Pizza

The models are saved in app folder

The model name should be plurasized e.g if the table is recipes call the model php artisan make:model recipe
