URL Wildcards

Each item in the database will have an id

The id of the item will form part of the url

pizzahouse.com/pizzas/1

pizzahouse.com/pizzas/2

pizzahouse.com/pizzas/3

pizzahouse.com/pizzas/4

or

pizzahouse.com/pizzas/pepperoni

pizzahouse.com/pizzas/volcano

pizzahouse.com/pizzas/mightymeaty

The final part of the url is the id

pizzahouse.com/pizzas/{id}


To use wildcards needs to set them up in the route.
